# Human Activity Recognition System

## Description
This project detects and classifies human activities from video using a deep learning model.  
The system can recognize different human movements such as walking, running, and standing.

---

## Features
- [x] Human detection from video
- [x] Activity classification using trained model
- [x] Real-time prediction display
- [ ] Additional activity classes (future work)

---

---

## Requirements
- Python 3.x
- OpenCV
- NumPy
- TensorFlow / Keras

Install dependencies:

```bash
pip install opencv-python numpy tensorflow
```
---

## How to Run
- Train model

```bash
python train_model.py
```

- Run activity recognition
  
```bash
python main.py
```
